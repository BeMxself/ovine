## @ovine/craft UI 编辑器

UI 通过点选的方式简化 JSON 配置过程，通过点选关联数据的方式，快速实现页面编辑的工具。

### PLAN:

- 添加 简单 保存功能
- 添加 简单的代码编辑功能
- 完善 组件类型（着重处理特殊情况）
- 完善 请求/权限/弹窗 等关联关系
- 尝试编辑初一基础页面

### schmea 原则

- 配置可扩展性最强
  - 能使用 Array 地方均使用 Array
  - 能扩展的地方全部扩展---比如，title: xxx ==> 可扩展为 title: [{type: 'html', html: 'xx'}]
- 尽可能优化 复制/剪切/粘贴 等操作
